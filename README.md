📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a housing dataset to understand the factors that influence house prices. The goal is to explore the dataset, identify patterns, detect outliers, and understand relationships between different variables affecting housing prices.

EDA helps in data understanding and feature selection before building machine learning models.

📊 Dataset

The dataset contains information about houses such as location, size, number of rooms, and price.

Features

price – Price of the house

area – Total area of the house (square feet)

bedrooms – Number of bedrooms

bathrooms – Number of bathrooms

stories – Number of floors

parking – Parking availability

mainroad – Whether the house is connected to the main road

guestroom – Availability of guest room

basement – Presence of basement

airconditioning – Air conditioning availability

prefarea – Preferred area

furnishingstatus – Furnishing status (furnished/semi-furnished/unfurnished)

🎯 Objective

Understand the structure of the housing dataset

Identify patterns and relationships between variables

Detect missing values and outliers

Visualize key insights using graphs and charts

🔍 Exploratory Data Analysis Steps

The following EDA steps were performed:

Data inspection using .head(), .info(), .describe()

Checking missing values

Correlation analysis between features

Outlier detection

Feature distribution analysis

📈 Data Visualization

Different visualizations were used to analyze the data:

Histogram – Distribution of numerical variables

Boxplot – Outlier detection

Scatter plot – Relationship between area and price

Heatmap – Correlation between variables

Countplot – Frequency of categorical features

Visualization libraries used:

Matplotlib

Seaborn
